## Customization

`RGSDotSequenceView` can be customized via the following @properties

```swift
var length: Int

var inactiveDiameter: CGFloat
var inactiveColor: UIColor

var activeScale: CGFloat
var activeColor: UIColor

var spacing: CGFloat

func next()
func pre()
```
